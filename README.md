# mcko

Локальный AI-ассистент с глобальными хоткеями и офлайн-кэшем Python-зависимостей.

## Установка

```bash
git clone <REPO_URL>
cd mcko
bash run.sh
```

`run.sh`:
- проверяет системные зависимости;
- ставит Python-пакеты из локальной папки `packages/`;
- при первом запуске создает `.env` из `.env.example`;
- запускает приложение через `watchdog.sh`.

## Первый запуск

После первого `bash run.sh` открой `.env`, укажи `OPENROUTER_API_KEY` и запусти `bash run.sh` еще раз.

Ключ в публичный репозиторий не включается.

## Требования

- Linux
- `python3`
- пакетный менеджер `apt-get` или `dnf`
- доступ к `sudo` для установки системных пакетов
